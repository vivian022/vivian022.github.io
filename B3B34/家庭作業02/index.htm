日本神準愛情心理測驗！6道題看出你在戀愛中的個性、會愛上什麼人

一個人潛意識中的選擇其實代表了你在愛情中的個性，甚至是會喜歡的類型，日網ドレス就公布了一則愛情心理測驗，只要答出以下六道題，就可以看出你的愛情觀

Q1.你展開了新生活，並且獨自住在陌生的城市裡。你第一個發訊息的人會是誰呢？

  1. 爸爸
  2. 媽媽
  3. 戀人
  4. 朋友
  • 解析
    • 在心理學中心生活代表的「期待和不安混雜在一起的情況」，訊息的意思是「連結和依靠」。因此從你想要傳訊息的人可以看出，你渴望被愛的強烈程度。
    1. 選擇爸爸，你渴望被愛的強烈程度佔了大約40%。你是一個不太會黏人的人，在戀愛時希望和對方是平等的關係，並且你在心靈上、精神上都是一個比較獨立的人。
    2. 選擇媽媽，代表你對於被愛的渴望有60％，並且是善於撒嬌的人（善於不等於喜歡、常做）。實際上這些表現可能代表，你在感情中是那種總是希望對方可以按照你的想法去做的類型，但是你並不會表現在臉上。雖然在撒嬌著要求，但卻讓人感覺不到你在逼迫，可以說是很厲害的戀愛高手。
    3. 選擇戀人，被愛的渴望佔了80％，在感情中非常黏人。比起為對方付出，更希望對方願意為你付出，雖然黏人、愛撒嬌的你可能很可愛，但是要小心不要為了取得另一半的注意力，而做出太超過的行為，不然最後很可能造成反效果，導致另一半受不了。有時適時的給彼此一點空間才能幫助感情更長久。
    4. 選擇朋友，你對於被愛的渴望是20％，而且你不是一個善於依靠的人。比起希望對方為你付出，你是自己會先為對方盡心盡力的類型。但是一段感情中如果只有一方在努力，也會變得疲憊，有時試著學習依賴撒嬌、愛自己多一點，在戀愛中會更耀眼、更受人喜愛。

    Q2.你在路邊看到一隻流浪貓，於是你拿了一點麵包給牠，接著貓咪喵了一聲，你覺得？

    1. 牠在對你說謝謝
    2. 希望你帶牠回家
    3. 懇求你不要扔掉牠
    4. 你想把牠交給中途、幫牠找領養人
  • 解析
    • 在心理學層面來說，被遺棄的流浪貓代表的是「對另一半的執著及依戀」的意思。從你選擇的答案中，可以看出你對另一半的依戀程度。
    1. 選擇牠在對你說謝謝，代表你在感情中是一個容易委屈自己配合另一半，或是因為害怕對方不喜歡你，而勉強自己迎合對方的人。但是一直委屈自己反而會讓你在感情中疲憊，有時候試著表達出自己真實的心意可能會更好。
    2. 選擇希望你帶牠回家，你在感情中比較強勢，認為如果另一半愛你，那他就要照著你的步調生活。但是一個人被限制久了，也會感到厭煩，因此也要偶爾顧及另一半的感受才是長遠之道。
    3. 選擇懇求你不要扔掉牠，你常常利用撒嬌、讓對方感到愧疚的方法，促使另一半將生活重心、注意力放在你身上，因為你每次這樣做時，他就會滿足你。雖然在感情中兩個人應該是單單看著對方，但是千萬不要為了綁住對方而過猶不及。情更長久。
    4. 選擇你想把牠交給中途、幫牠找領養人，雖然表面上看不出來，但是你其實是最依賴另一半的人。你總是用「冷漠」來確定另一半對你的心意，希望對方可以先關心你的感受、總想確認另一半是不是愛你，但是這樣確認愛情的方式並不健康，如果能坦誠的說出你的感受會更好。

    Q3.在夢裡，你喜歡的人有一部的身體消失了，你認為是哪裡？

    1. 眼睛
    2. 手
    3. 腳
    4. 心臟
  • 解析
    • 這個心理測試可以看出你的「獨佔欲」。身體在心理學中意味著對外界的刺激，失去的身體部位代表你執著的地方。
    1. 選擇眼睛，代表你在戀愛中的佔有慾比較強，希望對方一直指注視著你就好。只要另一半有一點心思不在你身上，你就會受不了、忌妒。但是這樣的關係讓你們彼此都喘不過氣，如果能夠放手一點，那你們的關係會變得更好。
    2. 選擇手，代表你很容易陷入肉體的情慾關係、不小心暈船。在你感到孤獨時，往往會覺得只要對方給你一點關注、溫柔那就是愛，但是這樣的感情卻很有可能讓你受傷！
    3. 選擇腳，是想要獨佔另一半的「自由」。如果你知道他一天的行程或去向，你就會感到不安，雖然你並不是想要束縛他，但是這樣很容易讓對方感到被監視。
    4. 選擇心臟，代表你是婚後佔有慾高的人。和另一半結婚後，你會希望能獨佔對方，但是如果你們兩人對於婚姻的定義不一樣，很可能會造成對方的痛苦，這時暫時停在交往的階段可能比較好。

    Q4.如果今天你要養一隻鳥，你會選擇什麼顏色的？

    1. 天藍色
    2. 黃色
    3. 黑色
    4. 白色
  • 解析
    • 在空中飛翔的小鳥在代表的是「戀愛」的意思，而你選擇的顏色則表示你會被哪種類型的人吸引。
    1. 選擇天藍色，會被有才華的壞男人吸引，因為對方很有才華，因此就算是不工作當米蟲、向你要生活費，你還是會為他著迷。
    2. 選擇黃色，一開始會被對方天真開朗的外表所吸引，但是你也有可能遇到外表開朗，實際上非常情緒化的男人。
    3. 選擇黑色，在感情上很容易被道德綁架。剛開始因為對方的熱情所吸引，但是後來就算發現對方會語言傷害、有暴力傾向，你也會因為他的一句道歉而與他和好。
    4. 選擇白色，更容易被另一半掌控，雖然剛開始對方以真誠、堅定的心打動你，但是他卻對你的掌控欲非常強烈，甚至到沒有自由的地步。

    Q5.今天你的房間出現了一隻蒼蠅，你會怎麼做？

    1. 用手揮打
    2. 用殺蟲劑
    3. 用蒼蠅拍
    4. 打開窗戶
  • 解析
    • 許多人不喜歡蒼蠅這種生物，而這其實代表了「人對於陌生事物的自卑感」，因此從如何處理房間裡的蒼蠅可以看出你面對自卑情結的方法。
    1. 選擇用手揮打，你是面對自卑感會想要克服的類型。如果有自己不足的地方，就會去學習，但是有時也會對自己過於嚴格。不要只是想著消除自卑感，而是把它當成「自己的特點」來生活。人因不完美而美麗。
    2. 選擇用殺蟲劑，你是習慣逃避自卑的類型。每次總是想著「今年一定要努力克服」，最後卻總是這樣想著過了一天。也許是想逃避自己不足的强烈心情，如果能更加確立自己的目標與現狀的差異，就更能克服問題。
    3. 選擇用蒼蠅拍，你面對自卑的地方，是習慣借助他人幫助的類型。但是如果將所有的事情都放手給別人，那你很有可能在面對問題時變得束手無策。
    4. 選擇打開窗戶，你是允許自己有自卑感的類型。雖然意識到自己的不足之處，但也不會去改變，而是接受「包括缺點在內的自己」。但是，另一方面也可能出現限制了自己可以突破的目標、無視問題的情况。

    Q6.你去了遊樂園，通常會最先玩什麼遊戲？

    1. 摩天輪
    2. 雲霄飛車
    3. 卡丁車/碰碰車
    4. 旋轉木馬
  • 解析
    • 在心理學中，遊樂園意味著設施豐富，是可以讓人恢復精神的地方。而其中的遊樂設施則表示不滿。
    1. 選擇摩天輪，你是在長期交往中，容易對對方感到不平等的類型。你在交往中容易累積不滿，但是卻不開口表達，因此容易造成雙方的隔閡，如果想要恢復關係的話，請兩人好好的溝通吧！
    2. 選擇雲霄飛車，你是沒辦法和同一個人交往很久的類型。度過熱戀期之後就很容易因為太平淡而感到不安，不過這時可以嘗試頏平時不同的相處模式，讓戀愛的火花再次燃起～
    3. 選擇卡丁車/碰碰車，你是容易對兩人的未來感到不安的類型，總是會想著要不要結婚、走下去，不過如果遇到這種不確定的時刻，先確認另一半的想法才能讓兩人對未來更有共識！
    4. 選擇旋轉木馬，你是交往一段時間後就會增加妄想的類型，會逐漸懷疑自己是不是沒有那麼愛對方，或是另一半對你的愛也不再珍惜，這種時候比起語言，更應該觀察對方的行動，找回剛交往時的溫暖。
